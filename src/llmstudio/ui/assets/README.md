# UI assets

Place the app **logo** here as one of:

```
logo.png   (recommended)
logo.svg
logo.jpg / logo.jpeg
logo.webp
```

The header automatically picks it up (first match wins, in the order above),
embeds it inline as a data URI, and also uses it as the browser-tab favicon.
If no file is present, the header falls back to the 🧪 emoji mark.

A roughly square or compact image reads best in the header (it's rendered ~58px
tall). Transparent PNGs are fine — they're shown on a small white tile.
